This project was a way for me to experiment in creating a web app utilizing React, Redux, and React-Redux. I used this project as a way to understand the effective layout of React-Redux and gain a better understanding of how it connects.

This is a purely front-end project. In the future I hope to have it hosted from a server, allowing authentication and shared session management, as well as storing user customizations and games

Javascript, React, Redux, React-Redux, HTML, CSS, and Bootstrap.
